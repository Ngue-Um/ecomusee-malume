# Ecomusée de Malume — Site GitHub Pages

Pack prêt à publier (Jekyll + Just the Docs) pour :
https://ngue-um.github.io/ecomusee-malume/

## Publier
1. Créez le dépôt `ecomusee-malume` sur GitHub (compte **Ngue-Um**).
2. Uploadez tout le contenu de ce pack à la racine du dépôt.
3. Settings → Pages → Source : **GitHub Actions**.
4. Attendez le déploiement, puis ouvrez l’URL.

## Ajouter vos images
Mettez vos photos dans `assets/img/` puis éditez `_pages/galerie.md`.
